Parte 2| Definio da funo que calcula a vario de tempo

A funo recebe dois pontos (x1, y1) e (x2, y2) que so usados para calcular os diferenciais de x e y.

Caso o diferecial de x seja menor que zero, impossvel fisicamente que um corpo a percorra a curva. Para isso, armazenado um valor na lista o qual podemos identificar futuramente que houve uma operao impossvel, e avisar ao usurio que h algum erro.

def deltaTempo (x1, y1, x2, y2, vel0):
    Dx = x2 - x1
    Dy = y2 - y1
    
    if Dx < 0:
        return "Delta X < 0!", ""
    
    teta = atan(Dy/Dx)
    L = (Dx**2 + Dy**2)**(1/2)
    v = (2*g*sin(teta)*L + vel0**2)**(1/2)
    
    dt = (v - vel0)/(g*sin(teta))
    
    return dt, v

def tempoTotal (f, var0, var1):
    lista_var = [var0]
    Dvar = var1 - var0
    
    for i in range(1, numDePontos + 1):
        lista_var.append(i*(Dvar/numDePontos) + var0)
        
    v0 = 0
    dts = [0]
    for i in range(len(lista_var) - 1):
        if i != var1:
            valor_t = f(lista_var[i])
            valor_t1 = f(lista_var[i+1])
            delta_t = deltaTempo(valor_t[0], valor_t[1], valor_t1[0], valor_t1[1], v0)
            
            dts.append(delta_t[0])
            v0 = delta_t[1]
            
    if str("Delta X < 0!") in dts:
        return "Delta X < 0!"
    
    else:
        T = sum(dts)
        return T

Parte 8| A cicloide a curva na qual um corpo solto em sua extremidade superior chega mais rpido extremidade inferior em comparao com qualquer outra curva existente.

Para comprovao, vamos definir 3 equaes: A cicloide, uma reta e uma funo de terceiro grau.

Em todas as curvas, o corpo ser solto da mesma posio, e chegar ao final da curva tambm na mesma posio.

  1. A cicloide ser gerada partir de uma circunferncia de raio 1m, ou seja, ter uma altura de 2m.
  2. A reta dever passar pelos pontos (0,0) e (pi, -2)
  3. A equao do polinmio de terceiro grau passar por trs pontos P1(0,0), P2(0.5, -1.5), P3(2,-1) e P4(pi,-2).
